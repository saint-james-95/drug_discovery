This package includes the following files:

apclusterunix64.so: Shared object module containing affinity propagation implementation for Linux (64-bit only) -- no MATLAB required.  Can be called by user software.

apcluster.h: Header file outlining module interface; #include with user software.

apclusterdemo_linux.c: short demo source code for Linux
